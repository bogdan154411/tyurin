Задание 15

По заданным сопротивлением трех резисторов r1 r2 r3, которые соединены параллельно, определить общее сопротивление.

Формула:1/r = 1/r1 + 1/r2 + 1/r3 

Обозначения и данные 
-r1 первый резистор (2)
-r2 второй резистор (4)
-r3 третий резистор (1)
-dnmr (переменная)
-nmr (переменная) r1 * r2 * r3
-result 

Изменяем начальную формулу на более удобную:
r1 * r2 * r3/ r1 * r2 + r1 * r3 + r2 * r3 

Подаставляем значения r1=2, r2=4,r3=1
Получаем число 0.571428597
Для подтверждения моих знаяений используем команду lldb
